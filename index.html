Day-1

In Html, div is a block level element having defaults styling associated with it. div is mainly used as conatainer for other Html elements.

In Html, Span is a inline level elements having own default styling it is also a used as conatainer and are not semantic tag.

In Html, class is denoted by dot(.) and are used for group of elements having same type of styling.

In Html, id is denoted by # symbol and are unique for each elements.

In Html, self closing tag are the tags which do not need to close. We do not need to write closing tag specifically.

Example :- <br /> , <img />

In Html, we have 6 heading tags all which have own styling ,lesser the number associated with higher the size <h1> </h1> is use as main heading while rest are use for subheadings.

In Html,element are categories as block level and inline elements based on their styling associated with them

Block elements are those elements which takes entire width of the page having top and bottom margin

Examples :- <h1> <div>

inline elements are those elements which takes the width of the contents only

Examples :- <img> <link>

In Html, em is more of like italic tag but use for emphasise the elements which make the good ranking to the page according to search engine optimization

In Html, strong is more of like bold tag but use for strong emphasise the elements which make the good ranking to the page according to search engine optimization

In Html, attribute are additional information of tag which tells more behaviour of that tag.

Example :- class, id, href, src and more ...
